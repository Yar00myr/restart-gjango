## How to upload :hammer:

1. Create virtual environment:
   ```powershell
   python -m venv .venv
   ```
   or
   ```powershell
   python3 -m venv .venv
   ```

2. Activate virtual environment(venv):
+ For Windows:
  ```powershell
  .\.venv\Scripts\activate.ps1    
  ```
+ For Linux / macOS:
  ```powershell
